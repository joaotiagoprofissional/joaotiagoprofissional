# Olá João Tiago aqui... 👋

#### `Desenvolvedor Front-End`

Desenvolvedor Front-End com foco em React e TypeScript, criando interfaces modernas, responsivas e orientadas à experiência do usuário. Atuo no desenvolvimento de aplicações com JavaScript (ES6+), consumo de APIs REST, componentização e organização de código escalável utilizando boas práticas do ecossistema React.
Desenvolvo projetos completos que envolvem desde a estruturação da interface até funcionalidades como gerenciamento de estado, responsividade, navegação entre páginas e interação com o usuário, sempre buscando performance e clareza na experiência.
Minha trajetória anterior como gerente e designer me proporcionou uma forte visão de negócio, organização e resolução de problemas. Hoje, aplico isso no desenvolvimento, construindo soluções que não são apenas funcionais, mas que fazem sentido para o usuário final.

Tecnologias: React.js, TypeScript, JavaScript, HTML, CSS, Git
Foco: Interfaces performáticas, UX e boas práticas de desenvolvimento
